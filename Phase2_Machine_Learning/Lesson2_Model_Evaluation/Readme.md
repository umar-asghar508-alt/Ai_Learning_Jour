# 📈 Machine Learning: Salary & Student Scores Prediction

This project focuses on implementing **Simple Linear Regression** to solve predictive problems. Beyond just building models, a major part of this project was learning and applying **Model Evaluation** techniques to ensure accuracy and reliability.

##  Project Overview
I worked on two distinct datasets to understand how a single feature impacts a target variable:
1.  **Experience vs. Salary:** Predicting an employee's salary based on their years of professional experience.
2.  **Study Hours vs. Scores:** Predicting a student's exam score based on the number of hours they spent studying.

##  Model Evaluation (Key Learning)
The core objective of this project was to move beyond training and focus on **evaluation**. I implemented the following metrics to validate the model's performance:
- **Mean Absolute Error (MAE):** To understand the average magnitude of errors.
- **Mean Squared Error (MSE):** To penalize larger errors and check variance.
- **R-Squared (R²) Score:** To determine how well the independent variables explain the variability of the dependent variable.

##  Conclusion
By applying evaluation metrics, I was able to confirm the model's high accuracy. This project helped me understand that a model is only as good as its evaluation results.
